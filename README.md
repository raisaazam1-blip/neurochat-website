# NeuroChat - Next-Gen Neural Intelligence

A modern, visually stunning landing page for a conversational AI platform with an interactive chat demo.

## Features

✨ **Modern Design**
- Animated gradient background with grid animation
- Glassmorphism UI components
- Smooth fade-in animations
- Responsive mobile design

💬 **Interactive Chat Demo**
- Real-time message sending
- Typing indicator animation
- Bot response simulation
- Support for Enter key to send messages

🎨 **Visual Effects**
- Animated background gradients
- Pulsing status indicator
- Message slide animations
- Hover effects on buttons
- Glowing effects on focus

📱 **Responsive**
- Works perfectly on desktop, tablet, and mobile
- Adaptive layout for smaller screens
- Touch-friendly interface

## Getting Started

### Option 1: View Locally
1. Download or clone this repository
2. Open `index.html` in your web browser
3. Start using the chat demo!

### Option 2: Deploy on GitHub Pages
1. Go to your repository settings
2. Scroll to "GitHub Pages" section
3. Select "main" branch as the source
4. Your site will be live at: `https://raisaazam1-blip.github.io/neurochat-website`

## Customization

You can easily customize the website:

### Change Colors
Edit the CSS color values in the `<style>` section:
- Primary color: `#64c8ff` (cyan)
- Accent color: `#fbbf24` (amber)
- Background: `#0a0e27` (dark blue)

### Change Text
- Update the hero heading and description
- Modify the button text
- Change the chat title

### Add More Features
- Connect to a real API
- Add user authentication
- Store chat history
- Add more pages
